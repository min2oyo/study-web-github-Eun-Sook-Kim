@keyframes

  •   @keyframes <이름> {
    <선택자> {
      <스타일>
    }
  }
    • animation-name 속성으로 어떤 animation을 사용할지 이름으로 구분
    • 웹 문서에서 animation을 여러 개 정의할 수 있음
    • <선택자>
      • 0%, 50%, 100%
      • from, to

animaition

  • animation: All;
  • animation-name: <keyframes 이름> | none;
    • @keyframes의 <이름> 입력
  • animation-duration: <시간>;
    • 기본값: 0
  • animation-direction: normal | reverse | alternate | alternate-reverse;
    • normal
      • 기본값
      • from -> to로 진행
    • reverse
      • to -> from로 진행
    • alternate
      • 홀수 번째: normal
      • 짝수 번째: reverse
    • alternate-reverse
      • 홀수 번째: reverse
      • 짝수 번째: normal
  • animation-iteration-count: <숫자> | infinite;
    • 숫자: 반복 횟수
    • infinite: 무한 반복
  • animation-timing-function: linear | ease | ease-in | ease-out | ease-in-out | cubic-bezier(n, n, n, n);
    • linear
      • 같은 속도
    • ease
      • 기본값
      • 천천히 -> 점점 빨리 -> 천천히
    • ease-in
      • 느리게 시작
    • ease-out
      • 느리게 끝
    • ease-in-out
      • 느리게 시작 -> 느리게 끝
    • cubic-bezier(n, n, n, n)
      • 베지에 함수를 정의해서 사용
      • 0 ~ 1 사이만 사용함
