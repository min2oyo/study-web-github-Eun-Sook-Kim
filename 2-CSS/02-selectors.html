일반

  • 나중에 작성한 스타일이 적용됨
  • 배경색, 배경 이미지는 상속 안 됨

연결 선택자(조합 선택자, 콤비네이션 선택자, 콤비네이션 셀렉터 등)

하위 요소

  • 특정 요소를 기준으로 그 안에 포함된 요소
  • 자식 요소
    • 현재 요소를 기준으로 바로 한 단계 아래 요소
  • 손자 요소
    • 자식 요소의 한 단계 아래 요소

- descendant selector(하위 선택자)

section p {}
  • 상위요소 하위요소
  • 부모 요소에 포함된 모든 하위 요소에 스타일 적용
  • 자손 선택자라고도 함

- child selector(자식 선택자)

select>p {}
  • 부모요소>자식요소
  • 자식 요소에만 스타일 적용

형제 요소

  • 웹 문서에서 부모 요소가 같음
  • 형 요소
    • 먼저 나오는 요소
  • 동생 요소
    • 나중에 나오는 요소

- adjacent selector(인접 형제 선택자)

h1+p {}
  • 요소1+요소2
  • 요소1과 요소2는 같은 레벨이면서 요소1 이후 가장 먼저 오는 요소2 선택

- sibling selector(형제 선택자)

h1~p {}
  • 요소1~요소2
  • 모든 형제 요소에 적용

속성 선택자

  • 태그 안에서 사용하는 속성값에 따라 요소를 선택
  • 속성 값의 조건에 따라 특정 부분만 선택하기 쉬워서 상황에 맞는 스타일을 지정하기 쉬움

[속성] 선택자

  • 해당 속성 존재
a[href] {}
  • a 요소 중에 href 속성이 있는 요소
[required] {}
  • required 속성이 있는 요소

[속성 = 속성값] 선택자

a[target = _blank] {}
  • 해당 속성값 존재
  • a 요소 중에 target 속성값이 _blank인 요소

[속성 ~= 속성값] 선택자

[class ~= button] {}
  • 해당 속성값 포함
  • class값 중에 button이 있는 요소
  • 주의
    • 단어별
    • '-'을 연결한 단어에 스타일을 적용하지 않음
    • 'flat-button', 'buttons' 안 됨

[속성 |= 속성값] 선택자

a[title |= us] {}
  • 해당 속성값 포함
  • title 속성값에 us가 있거나 us-로 연결된 속성값이 있는 a 요소
  • 주의
    • 단어별
    • '-'을 연결한 단어에 스타일을 적용함

[속성 ^= 속성값] 선택자

a[title ^= eng] {}
  • 해당 속성값 시작
  • title 속성값이 eng로 시작하는 a 요소

[속성 $= 속성값] 선택자

[href $= xls] {}
  • 해당 속성값 끝
  • 링크한 파일 이름의 마지막 단어가 xls인 요소

[속성 *= 속성값]

[href *= w3]
  • 해당 속성값 포함
  • href 속성값 중에 'w3'가 포함된 요소

가상 클래스

  • 웹 문서의 여러 요소 중 원하는 요소 선택

- 사용자 동작

  • ① :link ② :visited ③ :hover ④ :active
    • 순서가 바뀌면 스타일 적용 제대로 안 됨
    • 메뉴 링크에서 자주 사용
  • :link
    • 방문 안 한 링크
  • :visited
    • 방문한 링크
  • :hover
    • 마우스를 올려놓았을 떄
  • :active
    • 클릭 했을 때
  • :focus
    • 커서 초점

- 요소 상태

  • 앵커
    • 특정 위치 이동

  • :target

    #intro:target {}
    • 앵커 대상에 스타일 적용

  • :enabled

    • 해당 요소가 사용할 수 있는 상태

  • :disabled

    • 해당 요소가 사용할 수 없는 상태

  • :checked

    #signup input:checked+label {}
    • checked 속성이 있는 요소

  • :not

    #signup input:not([type=radio]) {}
    • 괄호 안에 있는 요소 제외

구조 가상 클래스

  • 웹 문서의 구조를 기준으로 특정 위치에 있는 요소
  • class나 id를 사용하지 않고도 스타일을 지정할 요소가 몇 번째인지를 따져서 스타일을 적용할 수 있음

- 특정 위치

  • :only-child

    • 부모 안에 자식 요소가 하나뿐일 때 자식 요소 선택

  • A:only-type-of

    • 부모 안에 A요소가 하나뿐일 때 선택

  • :first-child

    • 부모 안에 있는 모든 요소 중 첫 번째 자식 요소

  • :last-child

    • 부모 안에 있는 모든 요소 중 마지막 자식 요소

  • :nth-child(n)

    • 부모 안에 있는 모든 요소 중 n 번째 자식 요소

  • :nth-last-child(n)

    • 부모 안에 있는 모든 요소 중 끝에서 n 번째 자식 요소

  • A:first-of-type

    • 부모 안에 있는 A 요소 중 첫 번째 자식 요소

  • A:last-of-type

    • 부모 안에 있는 A 요소 중 마지막 자식 요소

  • A:nth-of-type(n)

    • 부모 안에 있는 A 요소 중 n 번째 자식 요소

  • A:nth-last-of-type(n)

    • 부모 안에 있는 A 요소 중 끝에서 n 번째 자식 요소

- 수식 위치

  • div p:nth-child(3) {}

    • div 요소에서 세 번째 자식인 p요소에 스타일 적용

  • div p:nth-child(odd) {}, div p:nth-child(2n+1) {}

    • div 요소에서 홀수 번째 자식인 p요소에 스타일 적용
    • 반복된 규칙 사용
    • n은 0부터 시작

  • div p:nth-child(odd) {}, div p:nth-child(2n) {}

    • div 요소에서 짝수 번째 자식인 p요소에 스타일 적용

가상 요소

  • 문서 안의 여러 요소 중 특정 부분 선택
  • 특별히 화면에 보이는 부분을 꾸밀 때 불필요한 태그를 사용하지 않음

  • ::first-line

    • 지정한 요소의 첫 번째 줄

  • ::first-letter

    • 지정한 요소의 첫 번째 글자
    • 첫 번째 글자는 반드시 첫 번쨰 줄에 있어야 함

  • ::before

    • 지정한 요소의 내용 앞에 스타일 적용

  • ::after

    • 지정한 요소의 내용 뒤에 스타일 적용
