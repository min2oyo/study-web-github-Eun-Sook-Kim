transition

  •   .transition {
    transition: 2s ease-in;
    /*
      transition-property: 기본값 all;
      transition-duration: 2s;
      transition-timing-function: ease-in;
      transition-delay: 0;
    */
  }
  
  .transition:hover {
    width: 200px;
    height: 200px;
    background-color: #f50;
    transform: rotate(270deg);
  }
    • transition: <transition-property> | <transition-duration> | <transition-timing-function> | <transition-delay>;
    • 속성 순서 상관없음
    • 시간 속성이 2개이면 transition-duration, transition-delay순으로 적용

transition-property

  • transition-property: all;
    • all 값을 사용하거나 transition-property를 생략할 경우 요소의 모든 속성이 트랜지션 대상이 됨
    • 기본값
  • transition-property: none;
    • 트랜지션을 하는 동안 아무 속성도 바뀌지 않음
  • transition-property: <속성이름>;
    • 트랜지션 효과를 적용할 속성 지정
    • 배경색만 바꿀 것인지, width값을 바꿀 것인지 원하는 대상만 골라서 지정
    • 속성이 여럿일 경우 쉼표로 구분
  • transition-property: background-color;
    • transition-property: width, height;

      transition-duration

      •   .box {
    transition-property: width, height;
    transition-duration: 2s, 1s;
  }

  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 실행할 시간 지정
        • second(초)나 millisecond(밀리초)로 지정
        • 대상 속성이 여러 개라면 쉼표로 구분
        • 음수값은 0으로 간주
        • transtion 속성에서 지정한 개수와 transition-duration 속성에서 지정한 개수가 다를 경우 순서대로 시간값을 반복해서 적용함

      transition-timing-function

      • linear
        •   .box {
    transition-timing-function: linear;
    transition-duration: 3s;
  }

  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 시작부터 끝까지 똑같은 속도로 진행
      • ease
        •   .box {
    transition-timing-function: ease;
    transition-duration: 3s;
  }
  
  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 천천히 -> 점점 빨리 -> 천천히
        • 기본값
      • ease-in
        •   .box {
    transition-timing-function: ease-in;
    transition-duration: 3s;
  }

  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 느리게 시작
      • ease-out
        •   .box {
    transition-timing-function: ease-out;
    transition-duration: 3s;
  }
  
  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 느리게 끝
      • ease-in-out
        •   .box {
    transition-timing-function: ease-in-out;
    transition-duration: 3s;
  }

  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 느리게 시작 -> 느리게 끝
      • cubic-bezier
        •   .box {
    transition-timing-function: cubic-bezier;
    transition-duration: 3s;
  }
  
  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 베지에 함수를 정의해서 사용
        • 0 ~ 1 사이만 사용함

      transition-delay

      •   .box {
    transition-property: width, height;
    transition-duration: 2s, 1s;
    transition-delay: 1s;
  }

  .box:hover {
    width: 200px;
    height: 120px;
  }
        • 실행할 시간 지정
        • second(초)나 millisecond(밀리초)로 지정
        • 대상 속성이 여러 개라면 쉼표로 구분
        • 음수값은 0으로 간주
        • transtion 속성에서 지정한 개수와 transition-duration 속성에서 지정한 개수가 다를 경우 순서대로 시간값을 반복해서 적용함
