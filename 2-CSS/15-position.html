position

  • position: static;

    static
    • 문서의 흐름에 맞춰 배치
    • 'left, right, top, bottom' 사용 못함
    • 기본값

  • position: relative;

    relative1
    relative2
    • 'left, right, top, bottom' 사용 가능

  • position: absolute;

    absolute1
    absolute2
    • relative 값을 사용한 상위 요소를 기준으로 위치 지정

  • position: fixed;

    fixed
    • 브라우저 창을 기준으로 위치 지정
    • 스크롤을 내려도 위치 고정
