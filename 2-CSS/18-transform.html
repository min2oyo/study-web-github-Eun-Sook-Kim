translate

  • transform: translate(5px, 10px);
    • y값이 없으면 0으로 간주
  • transform: translate3d(5px, 10px, 15px);
  • transform: translateX(5px);
  • transform: translateY(5px);
  • transform: translateZ(5px);

scale

  • transform: scale(1.2, 1.1);
    • 배수
    • scale(2, 2) == scale(2)
  • transform: scale3d(1.5, 1.2, 1.3);
  • transform: scaleX(1.2);
  • transform: scaleY(1.2);
  • transform: scaleZ(1.2);

rotate

  • transform: rotate(20deg);
    • degree(각도)나 radian(레디안) 사용
    • 1 radian = 180º/π
    • 양수: 오른쪽 회전
    • 음수: 왼쪽 회전
  • transform: rotate(2, 3, 20deg);
    • x, y는 방향벡터
  • transform: rotate3d(2, 3, 4, 20deg);
    • x, y, z는 방향벡터
  • transform: rotateX(20deg);
  • transform: rotateY(20deg);
  • transform: rotateZ(20deg);

skew

  • transform: skew(5deg, 10deg);
    • 두 번째 값이 주어지지 않으면 y축은 0으로 간주
  • transform: skewX(10deg);
  • transform: skewY(10deg);

perspective

  • perspective: 200px;
    • 원근감
    • 부모 요소에 정의
    • 3차원 변형에서 사용
    • 사용하는 값은 0보다 커야할 것
    • 값이 클수록 사용자로부터 멀어짐
    • 원래 있던 위치에서 사용자가 있는 쪽으로 얼마나 이동하는지를 픽셀 크기로 나타냄
    • 원래 있던 위치에서 사용자가 있는 방향이나 반대 방향으로 잡아당기거나 밀어내어 원근감 표현
    • 회전 형태를 입체적으로 표현
