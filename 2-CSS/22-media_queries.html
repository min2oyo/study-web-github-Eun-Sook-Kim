구문

  • @media [only | not] 미디어 유형 [and 조건] * [and 조건] {}
    • 일반
      • screen이 아닌 미디어에서는 스크롤을 포함한 전체 문서를 height로 지정해야 함
      • print에서는 한 페이지 높이를 기준으로 함
    • [only | not]
      • @media screen and (min-width: 768px) and (max-width: 1439px) { }
      • only
        • 미디어 쿼리를 지원하지 않는 웹 브라우저에서는 미디어 쿼리를 무시하고 실행하지 않음
      • not
        • not 다음에 지정하는 미디어 유형은 제외
        • not tv: tv를 제외한 미디어 유형에만 적용
      • and
        • 조건을 여러 개 연결해서 추가할 수 있음
    • 미디어 유형
      • @media screen { }
      @media print { }
      • all
        • 모든 미디어 유형에 사용
      • print
        • 인쇄 장치에서 사용
      • screen
        • 컴퓨터 스크린에서 사용
        • 스마트폰의 스크린도 포함
      • aural
        • 음성 합성 장치에서 사용
        • 음성 합성 장치: 주로 화면을 읽어 소리로 출력해 주는 장치
      • braille
        • 점자 표시 장치에서 사용
      • handheld
        • 패드(pad)처럼 손에 들고 다니는 장치를 위한 CSS를 정의
      • projection
        • 프로젝터를 위한 CSS를 정의
      • tty
        • 디스플레이 기능이 제한된 장치에 맞는 CSS를 정의
        • 이런 장치에서는 픽셀(px) 단위를 사용할 수 없음
      • embossed
        • 점자 프린터에서 사용할 CSS를 정의
    • 조건
      • @medea screen and (min-width: 1440px) { }
      • width, height
        • 웹 페이지의 가로 너비, 세로 높이 지정
      • min-width, min-height
        • 웹 페이지의 최소 너비, 최소 높이 지정
      • max-width, max-height
        • 웹 페이지의 최대 너비, 최대 높이 지정
    • 단말기 속성
      • @medea screen and (min-device-width: 375px) and (min-device-height: 812px) { }
      • device-width, device-height
        • 단말기의 가로 너비, 세로 높이
      • min-device-width, min-device-height
        • 단말기의 최소 너비, 최소 높이
      • max-device-width, max-device-height
        • 단말기의 최대 너비, 최대 높이
    • 회전
      • @media screen and(min-device-width: 812px) and (orientation: landscape) {}
      • orientation: portrait
        • 단말기의 세로 모드
      • orientation: landscape
        • 단말기의 가로 모드
        • 기본값
    • 미디어 쿼리 중단점
      • mobile first 기법
        • 모바일 페이지 먼저 작업 후 태블릿, PC에 맞춰 나감
      • 스마트폰
        • 모바일 페이지는 미디어 쿼리를 사용하지 않고 기본 CSS로 작성
        • 스마트폰이 방향까지 고려해서 제작할 시
          • min-width 가로: landscape 480px
          • min-width 세로: portrait 320px
      • 태블릿
        • 세로 크기가 768px 이상일 때 티블릿으로 지정
        • 가로 크기는 데스크톱과 똑같이 1024px 이상으로 지정
      • 데스크톱
        • 화면 크기가 1024px 이상이면 데스크톱으로 설정
    • 파일 연결
      • @import보단 <link> 사용
      • 인터넷 익스플로러에서 js를 먼저 받고 @import를 받으므로 오류 날 수 있음
      <link rel="stylesheet" href="#" media="print">
      • <link rel="stylesheet" href="#" media="미디어 쿼리 조건">
      @import url("#") only screen and (min-width: 321px) and (max-width: 768px);
      • @import url(css 파일 경로) 미디어 쿼리 조건
    • style에 적용
      • <style media="screen and (max-width: 320px)">
          body {
              background-color: orange;
          }
      </style>
      <style>
          @media screen and (max-width: 320px) {
              body {
                  background-color: orange;
              }
          }
      </style>
