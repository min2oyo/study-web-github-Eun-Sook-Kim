  • border-width: 5px 10px;

    • top, bottom: 5px
    • left, right: 10px

  • border-width: 5px 10px 15px;

    • top: 5px
    • left, right: 10px
    • bottom: 15px

  • border-width: 5px 10px 15px 20px;

    • top: 5px
    • left: 10px
    • bottom: 15px
    • right: 20px

  • border-width: thin;

  • border-width: medium;

  • border-width: thick;

border-style

  • border-style: none;

    • 테두리 없음
    • 기본값

  • border-style: hidden;

    • 테두리 감춤
    • 표에서 'border-collapse: collapse'일 경우 다른 테두리도 표시 안 됨

  • border-style: solid;

    • 실선

  • border-style: dotted;

    • 점선

  • border-style: dashed;

    • 짧은 직선

  • border-style: double;

    • 이중선
    • 'border-width': 두 선 사이의 간격

  • border-style: groove;

    • 테두리 창에 조각한 것처럼 표시
    • 홈이 파인 듯한 입체 느낌

  • border-style: ridge;

    • 창에서 튀어나온 것처럼 표시

  • border-style: inset;

    • 표에서 'border-collapse: seperate'일 경우 전체 박스 테두리가 창에 박혀 있는 것처럼 표시
    • 표에서 'border-collapse: collapse'일 경우 groove와 똑같이 표시

  • border-style: outset;

    • 표에서 'border-collapse: seperate'일 경우 전체 박스 테두리가 창에서 튀어나온 것처럼 표시
    • 표에서 'border-collapse: collapse'일 경우 ridge 똑같이 표시
