form

- 속성

  • action=""
    • <form> 태그 안의 내용을 처리해 줄 서버 프로그램 지정
  • method=""
    • 사용자가 입력한 내용을 서버 쪽 프로그램으로 어떻게 넘겨줄 것인지 지정
    • get
      • 데이터를 256 ~ 4,096byte까지만 서버로 넘김
      • 주소 표시줄에 사용자가 입력한 내용이 그대로 나타남
    • post
      • 입력한 내용의 길이에 제한받지 않고 사용자가 입력한 내용도 드러나지 않음
  • name=""
    • 자바스크립트로 폼을 제어할 때 사용할 폼의 이름 지정
  • target=""
    • action 속성에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에서 열도록 함
  • autocomplete=""
    • 자동완성
    • on: 기본값
    • off

- 전화번호 입력

  • size="": 필드 길이
  • value="": 화면에 표시할 텍스트
  • minlength="": 텍스트 최소 길이
  • maxlength="": 텍스트 최대 길이
  • placeholder="": 힌트
  • list="": 연결할 데이터 목록 지정. datalist에 사용
  • autofocus: 마우스 포인터 자동 위치
  • required: 필수 입력
  • readonly: 읽기 전용
